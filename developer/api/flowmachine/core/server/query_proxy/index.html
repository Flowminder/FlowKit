Class QueryProxy¶

QueryProxy(query_kind, params, *, redis=None, func_construct_query_object=None)
Source: flowmachine/core/server/query_proxy.py

This class acts as the interface and "translator" between the JSON representation of a query (via query_kind and params) and the actual flowmachine.core.query.Query object. It is responsible for storing the lookup between query_id and a string representing (query_kind, params) in redis and for constructing the flowmachine.core.query.Query if needed.
